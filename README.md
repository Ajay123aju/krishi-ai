# KRISHI AI 🌾

## Overview
AI-powered intelligent farming assistant for small and medium-scale farmers.

## Problem Statement
Farmers often face delayed crop disease detection, poor access to agricultural guidance, and missed crop management schedules.

## Proposed Solution
KRISHI AI provides:
- AI-powered crop guidance
- Image-based crop issue analysis
- Smart irrigation/fertilizer reminders
- Intelligent farming assistance

## Tech Stack
- Python
- Flask
- Gemini API
- SQLite
- HTML/CSS/JS
